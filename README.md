# Proyecto RAG-Odoo-Groq-IA

Este repositorio contiene un bot de Telegram integrado con APIs de Odoo y Groq IA, implementando un sistema de RAG (Retrieval-Augmented Generation) para respuestas contextuales.

## Estructura del proyecto

- **principal/**: Código principal del bot
- **manipadores/**: Handlers de Telegram
- **servicios/**: Servicios de IA y Odoo
- **configuración/**: Archivos de configuración
- **README.md**: Este archivo
- **requirements.txt**: Dependencias de Python
- **start_wrapper.sh**: Script de inicio

## Características

- ✅ Integración con Odoo ERP
- ✅ Conexión con Groq IA para generación de respuestas
- ✅ Sistema RAG para búsqueda contextual
- ✅ Bot de Telegram completamente funcional
- ✅ Configuración modular

## Requisitos previos

- Python 3.8+
- Cuenta de Telegram Bot (Token)
- Acceso a API de Groq
- Instancia de Odoo (opcional)
